Миграция данных: сверка систем учета оборудования

Автоматизация процесса верификации данных при переходе со старой системы учета на новую. Обеспечение корректности данных перед запуском системы в эксплуатацию.

Провести валидацию данных при переходе со старой системы учета (Система 1) на новую (Система 2). Обеспечить корректность переноса информации об оборудовании перед запуском системы в эксплуатацию.

  • Обеспечить целостность данных при миграции
  • Создать инструмент для регулярного мониторинга

1. Подготовка данных:
  • ВПР: Заполнение столбца "МОЛ склада" по справочнику
  • Текстовые функции: Разделение столбца "Оборудование" на название и код
  • Функция ЕСЛИ: Классификация оборудования на "Новое" и "б/у"
2. Анализ и визуализация:
  • Объединение данных двух систем
  • Создание сводных таблиц для сравнения

  • Автоматизирован процесс сверки данных между системами
  • Подготовлены данные для загрузки в новую систему
  • Снижены риски потери данных при миграции

Ключевые функции решения

  • Классификация оборудования
  • Сравнительный анализ систем
  • Гибкая система фильтров
  • Детальная отчетность

Примеры реализованных формул

 Заполнение МОЛ склада
=ВПР(C:C;'Справочник МОЛ'!A:B;2;0)

 Классификация состояния
=ЕСЛИ(I2>0;"б/у";"Новое")

 Разделение названия и кода
 Получение названия:
 =ПРАВСИМВ(F2;12)
 Получение кода:
=ЛЕВСИМВ(F2;ДЛСТР(F2)-12)
